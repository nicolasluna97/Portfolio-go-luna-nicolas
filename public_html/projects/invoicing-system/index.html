Sistema de Facturación (Multi-tenant) Invoicing System (Multi-tenant)

Plataforma de facturación e inventario con roles, clientes, productos, movimientos de stock y estadísticas para empresas o autonomos. La página web posse actulaización de stock en tiempo real además modo oscuro y claro. Actualmente en desarrollo en beta 2.0, cuando se sumen las funcionalidades faltantes, se subirá a producción. Multi-tenant invoicing and inventory platform with roles, customers, products, stock movements, and analytics for businesses and self-employed professionals. It provides real-time stock updates and supports both dark and light themes. Currently in beta 2.0; once the remaining features are completed, it will be released to production.

Angular NestJS PostgreSQL Docker

Módulos y funcionalidades Modules & features

Inventario Inventory

  • Permite crear productos mediante formularios modales, asignando categorías, stock inicial, precio de compra y múltiples precios de venta por producto. Allows creating products via modal forms, assigning categories, initial stock, purchase cost, and multiple selling prices per product.
  • Habilita edición masiva de productos seleccionados, con validaciones por campo y feedback visual de errores. Enables bulk editing of selected products, with per-field validations and visual error feedback.
  • Soporta eliminación segura de uno o varios productos mediante modales de confirmación para evitar acciones accidentales. Supports safe deletion of one or multiple products through confirmation modals to prevent accidental actions.

Ventas Sales

  • Ofrece un flujo de venta rápido, pensado para uso operativo, donde se seleccionan productos, cantidades y lista de precios. Provides a fast, operations-oriented sales flow where products, quantities, and a price list are selected.
  • Soporta múltiples precios por producto, permitiendo calcular totales según el precio aplicado en cada venta. Supports multiple prices per product, calculating totals based on the price applied to each sale.
  • Al confirmar la operación, descuenta automáticamente el stock, impactando en inventario en tiempo real. On confirmation, it automatically decreases stock, updating inventory in real time.

Estadísticas Analytics

  • Permite analizar resultados por día, semana, mes o año, mediante filtros de fecha dinámicos. Allows analyzing results by day, week, month, or year using dynamic date filters.
  • Muestra KPIs principales como ventas totales, productos vendidos y ganancias del período seleccionado. Shows key KPIs such as total sales, items sold, and profit for the selected period.
  • Integra gráficos para visualizar la evolución de la recaudación y facilitar la toma de decisiones. Includes charts to visualize revenue trends and support decision-making.

Clientes Customers

  • Permite registrar y administrar clientes, centralizando la información de contacto y relación comercial. Lets you register and manage customers, centralizing contact details and business relationship data.
  • Facilita la asociación de ventas a clientes, mejorando el seguimiento histórico de operaciones. Enables linking sales to customers, improving historical tracking of operations.
  • Ofrece un listado estructurado para consulta y gestión rápida desde el sistema interno. Provides a structured list for quick lookup and management inside the system.
